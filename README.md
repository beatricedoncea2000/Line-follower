# Line-follower

#### Components
- Arduino Uno
- Zip-ties
- Breadboard
- Wheels
- LiPo battery
- Wires
- QTR-8A sensor
- Ball caster
- Chassis
- L293D motor driver
- DC motors<br>

#### Task
Implement a robot which can follow straight lines, curved lines, all in a route. The calibration should be implemented with automatic motor movement. <br>
